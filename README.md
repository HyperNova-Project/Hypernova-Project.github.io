# 🚀 HyperNova Website

Projeto estudantil de foguetes e inovação científica.  
**Transformando ideias em voo** — montamos, testamos, erramos e melhoramos.  
Feito por estudantes e entusiastas em Sorocaba/SP.  

---

## ✨ Versão 2.0 — Full Revamp
Reformulação completa do site, incluindo acessibilidade, performance, SEO, PWA e organização de código.

### ♿ Acessibilidade
- Suporte a motion preferences (sem animações para quem prefere menos movimento)  
- Navegação por teclado com foco visível (Tab + Espaço)  
- Contraste melhorado no footer  
- Botões com mínimo de 44x44px para toque  
- Painel de acessibilidade com opções de alto contraste, espaçamento aumentado e **fonte [OpenDyslexic](https://github.com/antijingoist/open-dyslexic/releases)** para leitura inclusiva  

### 📱 Responsividade
- Novo breakpoint para tablets (768px–1024px)  
- Layouts otimizados para todos os tamanhos de tela  
- Menu hamburger corrigido em tablets  

### ⚡ Performance
- Lazy loading em vídeos e imagens  
- `srcset` em imagens responsivas  
- Compressão Gzip e cache via `.htaccess`  

### 🔍 SEO & Descoberta
- Meta tags Open Graph e Twitter Card completas  
- `robots.txt` e `sitemap.xml` adicionados  
- Google Analytics GA4 configurado (ID: G-VXWD0TY3Z9)  

### 📲 PWA
- Manifest completo para app instalável  
- Tema, ícones e atalhos configurados  

### 🛡️ Tratamento de Erros & Segurança
- Página 404 customizada  
- Cabeçalhos de segurança adicionados
